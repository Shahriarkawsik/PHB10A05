What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a cross-platform and language-independent interface that treats an HTML or XML document as a tree structure wherein each node is an object representing a part of the document. The DOM represents a document with a logical tree. Each branch of the tree ends in a node, and each node contains objects. DOM methods allow programmatic access to the tree; with them one can change the structure, style or content of a document. Nodes can have event handlers (also known as event listeners) attached to them. Once an event is triggered, the event handlers get executed.

How do you select an element from the DOM? 
                1. getElementById: 
                const element = document.getElementById('myId'); 
                2. getElementsByClassName (returns a live HTMLCollection):
                const elements = document.getElementsByClassName('myClass');
                3. getElementsByTagName (also returns a live HTMLCollection):
                const elements = document.getElementsByTagName('div');
                4. querySelector (returns the first matching element): 
                const element = document.querySelector('.myClass'); // For class 
                const element = document.querySelector('#myId'); // For ID 
                const element = document.querySelector('div'); // For tag
What is event delegation in the context of the DOM, and why is it usefull?

Event delegation is a technique where a single event listener on a parent element manages events for child elements. It's useful for improving performance, simplifying code, and automatically handling dynamically added elements, making event management more efficient.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using setAttribute, getAttribute, and removeAttribute methods. To change styles, access the style property of the element. For example, element.style.color = 'red'; updates the color, while element.setAttribute('src', 'image.jpg'); changes an attribute like the source of an image.
